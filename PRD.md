PRD - Alpha Fitness

1. Visão Geral do Projeto

Nome do Projeto

Alpha Fitness

Descrição

A Alpha Fitness é um website responsivo para uma academia fictícia. O objetivo do site é apresentar os serviços oferecidos, os planos de assinatura, a equipe de profissionais e fornecer ferramentas úteis para os visitantes, como uma calculadora de IMC.

⸻

2. Objetivo

Desenvolver um site moderno, responsivo e intuitivo que permita aos usuários conhecer a academia, seus serviços e entrar em contato facilmente.

⸻

3. Público-Alvo

* Jovens e adultos interessados em musculação.
* Pessoas que desejam emagrecer.
* Pessoas que buscam melhorar a saúde e o condicionamento físico.
* Iniciantes e praticantes avançados de atividades físicas.

⸻

4. Funcionalidades

Página Inicial

* Banner principal com imagem da academia.
* Frase de impacto.
* Botão para visualizar os planos.

Sobre a Academia

* História da empresa.
* Missão, visão e valores.

Planos

* Plano Básico.
* Plano Intermediário.
* Plano Premium.
* Comparação entre os benefícios de cada plano.

Treinos

* Hipertrofia.
* Emagrecimento.
* Condicionamento físico.

Equipe

* Apresentação dos profissionais.
* Especialidades de cada membro da equipe.

Calculadora de IMC

* Campo para peso.
* Campo para altura.
* Exibição automática do resultado.
* Classificação do IMC.

Contato

* Formulário para envio de mensagens.
* Informações de contato.
* Redes sociais.

⸻

5. Requisitos Funcionais

* O usuário deve visualizar informações sobre a academia.
* O usuário deve consultar os planos disponíveis.
* O usuário deve utilizar a calculadora de IMC.
* O usuário deve conseguir enviar uma mensagem pelo formulário de contato.
* O site deve funcionar em computadores e dispositivos móveis.

⸻

6. Requisitos Não Funcionais

* Interface responsiva.
* Navegação simples e intuitiva.
* Carregamento rápido das páginas.
* Código organizado e comentado.
* Compatibilidade com navegadores modernos.
